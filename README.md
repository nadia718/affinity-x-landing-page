# Affinity X Landing Page

A responsive landing page developed by converting a Figma design into a fully functional website using Bootstrap.

## Features

* Pixel-perfect Figma to Bootstrap conversion
* Fully responsive layout
* Modern and clean UI design
* Mobile-friendly navigation
* Optimized structure and reusable components
* Cross-browser compatibility

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

## Project Goal

The purpose of this project was to practice converting professional Figma designs into responsive and interactive web pages using Bootstrap.
